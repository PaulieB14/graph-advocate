## What it is

The Graph has 15,000+ subgraphs across 70+ networks. Finding the right one and writing a query that actually returns data normally takes 3–4 round trips of search-skim-test. Graph Advocate is **one endpoint that does it for you** — ask a plain-English question, get back the right subgraph, a runnable GraphQL query, and (when possible) the live answer.

It speaks A2A JSON-RPC, MCP SSE, and x402 payments. Any agent, human, or LLM can call it.

## See it work

<CodeGroup>

```bash One free call
curl -X POST https://graphadvocate.com/ \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc":"2.0","id":"1","method":"message/send",
    "params":{"message":{"messageId":"q","role":"user",
      "parts":[{"kind":"text","text":"Uniswap V3 pools on Base"}]}}
  }'
```

```json Response (truncated)
{
  "recommendation": "subgraph-registry",
  "confidence": "high",
  "query_ready": {
    "tool": "execute_query_by_subgraph_id",
    "args": {
      "subgraph_id": "HMuAwufqZ1YCRmzL2SfHTVkzZovC9VL2UAKhjvRqKiR1",
      "gql": "{ pools(first: 10, orderBy: totalValueLockedUSD) { ... } }"
    }
  },
  "execution_result": { "data": { "pools": [ ... ] } }
}
```
