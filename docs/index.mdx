## What it does

You ask in plain English. You get back **the right subgraph, a ready-to-run GraphQL query, and an MCP install hint**.

Graph Advocate is an **A2A-protocol, ERC-8004-registered, x402-paid** agent that routes onchain data questions to the right Graph service:

- **token-api** — balances, swaps, NFTs, holders (EVM + Solana + TON)
- **subgraph-registry** — 15,500+ indexed subgraphs
- **substreams** — raw block data, traces, streaming
- Protocol-specific MCPs — Aave, Polymarket, Limitless, Predict.fun, and more

Plus a new layer that nothing else has: **live x402 discovery** — join Paul's x402 Base subgraph, agent0 ERC-8004 registry, 8004scan, and the CDP Bazaar to show which services are actually being paid *right now*.
